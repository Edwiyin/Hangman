Jeu du Pendu en Go
Ce projet est une implémentation du jeu classique du Pendu en utilisant le langage de programmation Go. Il a été développé dans le cadre d'un projet visant à appliquer les concepts de base de l'algorithmie tout en respectant les bonnes pratiques de développement logiciel.
Fonctionnalités

Lecture de fichiers de mots pour différents niveaux de difficulté
Sélection aléatoire d'un mot à deviner
Révélation de lettres aléatoires au début du jeu
Interface utilisateur en ligne de commande avec menu interactif
Affichage graphique du pendu
Gestion des lettres déjà proposées
Différents niveaux de difficulté (Facile, Moyen, Difficile)
